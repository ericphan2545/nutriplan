Part 1. Cài đặt NPM Package, Tailwind CSS và Vite


NPM viết tắt của "Node Package Manager", là trình quản lý gói cho các ứng dụng JavaScript và Node.js.


 


Các em sẽ được học kỹ về các công cụ này ở phần tiếp theo của khoá học, ở bài này cứ cài đặt và hình dung được cách sử dụng là được.
	

Đầu tiên tạo file .gitignore (để ý dấu chấm đầu tên, đây gọi là Dot File). Với nội dung như sau 
# Logs
logs
*.log
npm-debug.log*
yarn-debug.log*
yarn-error.log*
pnpm-debug.log*
lerna-debug.log*


node_modules
dist
dist-ssr
*.local


# Editor directories and files
.vscode/*
!.vscode/extensions.json
.idea
.DS_Store
*.suo
*.ntvs*
*.njsproj
*.sln
*.sw?
	

Mục đích là để tránh commit nhầm những file thư viện sau này khi cài đặt NPM Packages


Tiếp theo, mở Terminal và chạy lệnh sau 
npm init
	Lệnh này để khởi tạo file package.json để quản lý các thư viện cần thiết. Khi chạy lệnh, sẽ hỏi nhập thông tin, các bạn nhập tuỳ theo ý mình.






Tiếp theo cài đặt các thư viện cần thiết, chạy lệnh sau vào Terminal
npm install -D tailwindcss @tailwindcss/postcss postcss autoprefixer vite
	

Tạo file postcss.config.js với nội dung sau 
export default {
  plugins: {
    "@tailwindcss/postcss": {},
    autoprefixer: {},
  },
}
	

Tạo file tailwind.config.js với nội dung sau
/** @type {import('tailwindcss').Config} */
export default {
  content: [
    "./index.html",
    "./src/**/*.{js,ts,jsx,tsx}",
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}
	

Tiếp theo, mở file package.json lên. 


Đầu tiên, thay thế propeties scripts bằng đoạn sau 
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1",
    "dev": "vite",
    "build": "vite build",
    "preview": "vite preview"
  },
	

Sau đó thêm dòng sau, dưới propeties name 
"type": "module",
	Mục đích là thiết lập các lệnh để chạy trong quá trình phát triển dự án


Tới đây thử chạy lệnh sau: 
npm run dev
	

Sẽ được kết quả như sau
  



Lúc này thử mở trình duyệt lên và truy cập vào http://localhost:5174
sẽ thấy được trang của bạn


Tới đây có thể commit code lại, với message là “Setup NPM and Vite”

Part 4. Thêm các thẻ Metadata và tạo Favicon cho trang 


Các thẻ Metadata là để khi em copy link gửi cho bạn bè hoặc submit lên Google Index có thể đọc được Title, mô tả và hình ảnh từ Site của các em 
  

Ví dụ sau khi anh làm bước này, anh share URL web lên Zalo sẽ hiển thị được như trên hình.


Có thể sử dụng ứng dụng sau để tạo metatags
https://www.seoptimer.com/meta-tag-generator


( nhớ sửa lại theo thông tin của mình nhé )
<meta name="title" content="Quyen's Site">
<meta name="description" content="Hi, my name is Quyen Nguyen. I'm the Web Developer.">
<meta name="keywords" content="web, fullstack developer">
<meta name="robots" content="index, follow">
<meta http-equiv="Content-Type" content="text/html; charset=utf-8">
<meta name="language" content="English">
<meta name="author" content="Quyen Nguyen Huu">
	

chèn thẻ này vào trong thẻ head của file index.hml


Tiếp theo là thiết lập thêm các thẻ meta theo chuẩn Open Graph protocol để có thể chia sẽ lên mạng xã hội như Google, Facebook hay Twitter
https://css-tricks.com/essential-meta-tags-social-media/


thường thì sẽ như sau ( nhớ sửa lại theo thông tin của mình nhé )
  <meta property="og:title" content="Quyen's Site" />
  <meta property="og:url" content="/" />
  <meta property="og:image" content="/banner.avif" />


  <meta name="twitter:title" content="Quyen's Site">
  <meta name="twitter:description" content="Hi, my name is Quyen Nguyen. I'm the Web Developer.">
  <meta name="twitter:image" content="/banner.avif">
  <meta name="twitter:card" content="Quyen's Site">
	

Cuối cùng là tạo Favicon cho Website, là cái icon nhỏ trên Tab của trình duyệt. 
Có thể sử dụng ứng dụng này để tạo Favicon https://favicon.io/


  



Sau khi tạo xong, tụi em tải file về rồi giải nén hết file vào folder public và update metatag theo hướng dẫn từ trang web là được.
  

File tải về sẽ có nội dung như sau, hãy copy bỏ hết vào public folder nhé.
  

(Mấy file ngoài favicon.ico là sử dụng để làm icon trên Mobile khi tụi em sử dụng tính năng Add to Home Screen ấy)

Part 5. Cập nhật lại Workflow code để deploy Landing Page bằng Github Action


Mở file .github/workflows/static.yml lên và sửa lại nội dung như sau 
# Simple workflow for deploying static content to GitHub Pages
name: Deploy static content to Pages


on:
  # Runs on pushes targeting the default branch
  push:
    branches: ["main"]


  # Allows you to run this workflow manually from the Actions tab
  workflow_dispatch:


# Sets permissions of the GITHUB_TOKEN to allow deployment to GitHub Pages
permissions:
  contents: read
  pages: write
  id-token: write


# Allow only one concurrent deployment, skipping runs queued between the run in-progress and latest queued.
# However, do NOT cancel in-progress runs as we want to allow these production deployments to complete.
concurrency:
  group: "pages"
  cancel-in-progress: false


jobs:
  # Single deploy job since we're just deploying
  deploy:
    environment:
      name: github-pages
      url: ${{ steps.deployment.outputs.page_url }}
    runs-on: ubuntu-latest
    steps:
      - name: Checkout
        uses: actions/checkout@v4
      - name: Set up Node
        uses: actions/setup-node@v4
        with:
          node-version: 22
          cache: 'npm'
      - name: Install dependencies
        run: npm install
      - name: Build
        run: npm run build
      - name: Setup Pages
        uses: actions/configure-pages@v4
      - name: Upload artifact
        uses: actions/upload-pages-artifact@v3
        with:
          # Upload entire repository
          path: './dist'
      - name: Deploy to GitHub Pages
        id: deployment
        uses: actions/deploy-pages@v4
	

Về cơ bản là thêm một bước build project bằng vite và upload folder dist lên Github Page 


Xong rồi, chờ Github Action chạy xong, ra kiểm trang Github Page thôi.

Part 6. Tạo hiệu ứng hiển thị dần dần khi cuộn trang
Để làm cái này các em cần cài thư viện scrollreveal bằng lệnh sau 
npm install scrollreveal --save
	

sau đó tạo file src/scripts/scroll-reveal.js với nội dung như sau:
import ScrollReveal from 'scrollreveal'


export default function initScrollReveal(defaultProps = null) {
    const targetElements = [
        {
            element: ".banner-text",
            animation: {
                delay: 700,
                origin: window.innerWidth > 768 ? "left" : "bottom",
            },
        },
        {
            element: ".banner-cta",
            animation: {
                delay: 1000,
                origin: window.innerWidth > 768 ? "left" : "bottom",
            },
        },
        {
            element: ".section-title",
            animation: {
                delay: 300,
                distance: "0px",
                origin: "bottom",
            },
        },
        {
            element: ".section-content",
            animation: {
                delay: 500,
                distance: "0px",
                origin: "bottom",
            },
        },
    ];
    
    ScrollReveal({ reset: false });
    
    targetElements.forEach(({ element, animation }) => {
        ScrollReveal().reveal(element, Object.assign({}, defaultProps, animation));
    });
}
	

Mở file src/scripts/application.js và sửa lại nội dung như thế này 
import '../css/style.css'
import initScrollReveal from './scroll-reveal';


initScrollReveal();
